# WP AI Content Assistant

AI-powered content generation plugin for WordPress.

Built with clean architecture principles, REST API integration, and reusable service layer design.  
Designed for enterprise-grade WordPress environments.

---

## Overview

WP AI Content Assistant integrates AI capabilities directly into WordPress.  
It allows administrators and editors to generate content securely using a configurable AI provider.

This plugin demonstrates:

- Separation of concerns
- Service layer abstraction
- REST API integration
- Secure API handling
- WordPress coding standards compliance
- Extendable architecture
